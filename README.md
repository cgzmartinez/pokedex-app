# Pokédex-App

## Description:

This is a [simple web application](https://cgzmartinez.github.io/pokedex-app/) that uses data from a [Pokémon API](https://pokeapi.co/).

## Tech:

Technologies used to make this app include the following:
- HTML
- CSS
- Javascript
- Bootstrap
